About
bossran

RAN Online was a massively multiplayer online role-playing game developed by Min Communications, Inc.. After starting the first official service in Korea in July 2004, RAN Online continued to expand globally.

This server is based on BabyRan with some improved and updated files and security features. This is a free to play server and only for study purpose. Though if you like the game and would like to support our work you can make donations.

You can read more about donations in the top up tab.
